### 万字文档+大学生心里健康管理系统+Vue2+SpringBoot
## 项目采用技术：
架构: B/S、MVC
系统环境：Windows/Mac
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue2、ElementUI、JavaScript、CSS、node.js
采用M（model）V（view）C（controller）三层体系结构，MySQL数据库作为系统数据储存平台，实现了基于B/S结构的Web系统。

系统功能模块：前台使用HTML+Laiyui，后台使用vue+ElementUI
管理员端和用户端2个角色
首页：欢迎使用 大学生心里健康管理系统
修改密码：修改自己密码
个人信息：查看个人信息
用户管理：用户管理的增加、修改、删除、查询、列表、统计
评价管理：评价管理的增加、修改、删除、查询、列表、回复
健康知识类型管理：健康知识类型管理的增加、修改、删除、查询、列表
时间段管理：时间段管理的增加、修改、删除、查询、列表
通知类型管理：通知类型管理的增加、修改、删除、查询、列表
健康知识管理：健康知识管理的增加、修改、删除、查询、列表
通知管理：通知管理的增加、修改、删除、查询、列表
学生管理：学生管理的增加、修改、删除、查询、列表
心理老师管理：心理老师管理的增加、修改、删除、查询、列表以及重置密码
心理老师留言管理：心理老师留言管理的增加、修改、删除、查询、列表以及回复
心理老师收藏管理：心理老师收藏管理的详情、删除、查询、列表
心理咨询预约申请管理：心理咨询预约申请管理的列表、删除、查询、审核以及详情
试卷管理：试卷管理的增加、修改、删除、查询、列表
试题管理：试题管理的增加、修改、删除、查询、列表
考试记录：考试记录的列表、查看考试详情也就是历史记录、删除、查询、列表
错题本：错题本的增加、修改、删除、查询、列表
轮播图管理：轮播图管理的增加、修改、删除、查询、列表
前台页面：
首页：健康知识展示、心理老师展示、通知展示、
试题卷：试卷列表、考试
健康知识：健康知识查看
通知：通知查看
心理老师：心理老师查看、详情、留言
个人中心：个人中心维护
